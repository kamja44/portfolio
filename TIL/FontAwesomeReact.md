1. Font Awesome의 SVG 기반 아이콘을 활성화 시키기 위해 기본 패키지인 @fortawesome/fontawesome-svg-core를 설치한다.
   `npm i @fortawesome/fontawesome-svg-core`
2. Font Awesome 아이콘에 대한 패키지를 설치한다.
   무료로 제공되는 Solid, Regular, Brands 3개의 카테고리에 대한 패키지만 설치한다.
   `npm i @fortawesome/free-solid-svg-icons @fortawesome/free-regular-svg-icons @fortawesome/free-brands-svg-icons`
3. Font Awesome을 React 컴포넌트 형태로 사용할 수 있도록 해주는 @fortawesome/react-fontawesome 패키지를 설치한다.
   `npm i @fortawesome/react-fontawesome`
